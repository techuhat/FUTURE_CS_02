This repository contains the work completed for Task 2 of the Future Interns Cyber Security Internship.

Analyze system logs to detect suspicious activities such as brute force attacks, unauthorized login attempts, and abnormal network behavior.

- Kali Linux
- Linux Command Line
- Log Analysis Techniques

- Investigated brute force login attempts
